# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
|--------|-----------|
| main   | ✅ Yes    |
| < main | ❌ No     |

Only the `main` branch receives active security fixes. Users are encouraged to stay up to date.

---

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**.

Instead, please report it responsibly using one of the following methods:

### Preferred: GitHub Security Advisories
1. Go to the **Security** tab of this repository
2. Click **Report a vulnerability**
3. Provide:
   - Description of the issue
   - Steps to reproduce
   - Impact assessment (if known)
   - Any suggested remediation

### Alternative: Direct Contact
If GitHub Security Advisories are unavailable, contact:

- **Maintainer:** Adonis Jimenez  
- **Email:** security@yourdomain.com *(replace if applicable)*
